title: Creating a New Documentation Project
description: Quick guide to set up a project in RaaziSpace.
---

## Overview

Create a structured project to manage your docs.

## Steps

<Steps>
  <Step title="Go to your Space">
    Select a space from your dashboard to add a project.
  </Step>
  <Step title="Add New Project">
    Click on "Add Project", give it a name, set visibility, and save.
  </Step>
  <Step title="Start Adding Pages">
    Use the "New Page" button inside your project to start documenting!
  </Step>
</Steps>
